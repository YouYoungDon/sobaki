## 11. MVP 범위

**MVP에 포함**:
- 홈(룸) 화면 — react-konva + Lottie
- 지출 기록 (영수증 AI 파싱 포함)
- 캘린더 가계부 (기록 조회)
- 아이템 자동 생성 (절약 행동 → 아이템)
- 퀘스트 시스템 (기본 daily 퀘스트)
- 아이템 도감

**MVP 이후 확장**:
- 토스 거래내역 API 자동 동기화 (raw_data 재사용)
- 위치 기반 "근처 저렴한 가게" 추천 (locations 테이블 활용)
- 소셜 기능 (친구 룸 방문)
- 룸 테마 변경
- 주간/월간 절약 리포트
