# Upbit 투자 대시보드

이 프로젝트는 Upbit 공개 API를 사용해 실시간 시장 정보를 조회하는 간단한 투자 대시보드입니다.

## 실행 방법

1. 프로젝트 루트에서 Vite 개발 서버 실행:

```bash
npm run dev
```

2. 브라우저에서 다음 주소로 접속:

```
http://localhost:5173/upbit-dashboard.html
```

## 기능

- Upbit KRW 마켓 목록을 자동으로 로드
- 선택한 마켓의 실시간 티커 정보 표시
- 현재가, 가격 변동, 변동률, 24시간 거래대금 표시
- 빠른 새로고침과 기본 마켓 초기화

## 주의

- 개인 계정 정보는 이 예제에 포함되지 않습니다.
- Upbit 개인 API를 사용하려면 서버 측 JWT 서명 로직이 필요합니다.
