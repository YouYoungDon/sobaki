# Sobagi Design Rules

## Mood

Sobagi should feel like:
- Animal Crossing
- cozy diary
- healing space
- warm small room

---

# Keywords

- warm
- soft
- emotional
- cozy
- cute
- peaceful

---

# Character

Sobagi is the emotional center of the app.

The character should:
- react often
- feel alive
- show emotions immediately
- feel comforting

---

# UI Tone

UI should feel:
- rounded
- soft
- breathable
- warm

Avoid:
- dense layouts
- heavy charts
- fintech style
- overly modern corporate design

---

# Animation

Use subtle:
- bouncing
- floating
- blinking
- sparkle effects

Animations should be:
- lightweight
- fast
- responsive

---

# Home Screen

The home screen is the emotional core.

Priority:
1. Sobagi
2. Emotion
3. Atmosphere
4. Spending summary

NOT:
- analytics first
- charts first

---

# Emotional Direction

The app should never shame the user.

Even overspending should feel:
- understood
- comforted
- emotionally safe
